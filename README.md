# О проекте  
  
Coursearch - это мультисервисная система, которая:  
- Парсит курсы с внешних платформ через модульные провайдеры  
- Строит персонализированные рекомендации с использованием гибридного ML-пайплайна  
- Обеспечивает семантический поиск по курсам через векторные эмбеддинги  
- Генерирует объяснения рекомендаций через LLM-интеграцию  
  
Проект решает проблему информационного перегруза в EdTech:   
  
Вместо ручного перебора сотен курсов пользователь получает релевантную подборку с обоснованием "почему именно это"  
  
# Пайплайн рекомендаций  
  
**Content-Based**  
- Поиск похожих курсов через косинусное сходство эмбеддингов в ChromaDB  
  
**Like-Ranker v3**  
- Ранжирование кандидатов с помощью градиентного бустинга на основе неявных фидбэков (лайки, просмотры)  
  
**Re-Ranking + Explanation**  
- Финальная корректировка + генерация человеческого объяснения через LLM

# Запуск  
  
1. Клонирование репозитория  
  
```commandline  
git clone https://github.com/yalama34/coursearch.git  
cd coursearch  
cp .env.example .env  
#важно отредактировать .env.example под себя  
```  
  
2. Запуск в режиме разработки  
  
```commandline  
docker compose -f docker-compose.dev.yml up --build  
  
#запуск продакшен конфига (образы заранее собраны)  
docker compose up -d  
```
